## 2. 시나리오 (5~10줄)
* 이 프로그램은 당신이 러닝할때 더 빠르고 오래 달릴수 있도록 도와주는 시스템입니다.
* 가장 먼저 오늘 러닝 목표를 설정하는것으로 시작됩니다. 그다음에는 실제로 오늘 달린 거리와 시간을 기입하여 평균 페이스를 계산합니다. 그런다음 오늘 목표 거리를 달성했는지 확인시켜주고, 목표 페이스랑은 얼마나 차이가나는지 확인시켜줍니다. 그 후에 나의 런닝 등급을 매겨줍니다 런닝 페이스가 빠르다면 S급 등급을 매겨주고 매우 느리다면 F급 등급을 매겨줍니다. 또한 내일 런닝을 더 잘 뛸 수 있도록 동기부여가 되는 말을 해줍니다. 마지막으로 내일 런닝 목표를 세워주면서 끝이납니다.

4. 오늘의 러닝 등급 산정

## 🟩 [2차 과제: V1.0] 입출력 + 리스트 + 조건문 - 향후 작성 예정
### **✨2차 과제 내용:**
(예시)
  * 이번 과제는 런닝 사용자로부터 데이터를 입력받아 리스트에 저장하고 페이스 계산 후 결과값을 도출하는 시스템을 구축하는것입니다. 특히 `if-elif` 조건문과 `and/or` 논리 연산자를 활용하여 상황에 맞는 등급과 그에따른 동기부여를 해주고 목표 거리를 달성했는지 확인하여야합니다
  * input()으로 사용자의 이름, 사용자의 나이, 오늘 달린 거리, 달린 시간, 목표 거리를 입력받아 리스트에 저장.
  * 입력받은 정보를 토대로 페이스 계산
  * for 을 사용하여 최근 4번의 런닝 기록을 입력하고 리스트에 추가, 오늘 뛴 기록까지 리스트에 추가
  * if를 사용하여 가장 낮은 기록을 제거
  * 입력받은 것들을 토대로 평균 거리와 최대 거리를 구해서 저장
  * if-elif-else를 사용하여 목표 달성 여부와 평균 대비 성과, 등급, 동기부여, 내일 목표까지 설정해주는 기능 구현
  
### **🤖 AI 파트너십 과정**
(예시) 
1. **내용 1: 컨셉에 맞춰 코드 추천받기**
    * **프롬프트 요약:** "내가 정한 컨셉과 내가 준 조건에 맞춰서 코드를 짜야하는데 도와줘."
    * **적용 내용:** AI가 제안한 코드들을 참고하여 부족한부분이 있다면 더 채우고 수정하여 코드를 완성함.

2. **내용 2: 효율적인 리스트 활용법 자문**
    * **프롬프트 요약:** "파이썬 기초 문법 단계(리스트 → 조건문 → 반복문 → 함수/딕셔너리)에 맞춰, '러닝 도우미' 컨셉의 프로그램이 어떻게 발전할 수 있을지 4단계 기능 확장 로드맵을 제안해줘."
    * **적용 내용:** 단순 입출력을 넘어, **'기초 정보 입력 및 리스트 저장(1차) → 목표 달성 여부 판단, 등급 산정, 성과 평가(2차) → 여러 기록 입력 반복처리(3차) → 딕셔너리를 활용한 사용자 정보/결과(4차)'**로 이어지는 논리적인 확장 로드맵을 확립함.
    
### **🛠️ Troubleshooting & 기술 회고:**
(예시) 
1. **문제 1: 리스트 정렬이 안되는 현상**
    * **원인:** 'sort' 를 써야하는데 'short'로 오타가 났음
    * **해결:** 오타를 고쳐서 해결함.
2. **문제 2: 조건문 뒤에 자꾸 생기는 빨간 줄 (SyntaxError)**
    * **원인:** `if`와 `elif` 조건식 끝에 마법의 기호인 `:`(콜론)을 자꾸 빠뜨려서 발생함.
    * **해결:** 파이썬의 모든 조건문과 반복문 끝에는 항상 땡땡(`:`)이 들어가야 한다는 규칙을 학습하고 수정함.
